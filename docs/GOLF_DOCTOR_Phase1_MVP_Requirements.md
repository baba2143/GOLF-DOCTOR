**作成日:** 2026年3月9日

**ステータス:** Phase 1 MVP 確定

## 1. プロジェクト概要

### 1.1 サービス概要

| 項目 | 内容 |
|-----|------|
| **サービス名** | GOLF DOCTOR |
| **コンセプト** | ゴルファーとレッスンプロをマッチングし、動画診断による1to1指導を提供 |
| **Phase 1 スコープ** | 動画診断（非同期）のみ |
| **ターゲット規模** | ユーザー 500名 / プロ 30名 |

### 1.2 提供価値

- **ユーザー（ゴルファー）:** 場所・時間を問わず「上達のきっかけ」を得られる
- **プロ:** 空き時間を活用した「新たな収益源」を獲得

### 1.3 開発対象プラットフォーム

| プラットフォーム | 対象 |
|----------------|------|
| iOS / Android | ユーザーアプリ、プロアプリ |
| Web（PC） | 管理者用管理画面 |

## 2. ビジネス要件

### 2.1 収益モデル

| 項目 | 内容 |
|-----|------|
| **レッスン料** | プロが価格帯から選択（500〜5,000円） |
| **プラットフォーム手数料** | レッスン料の **20%** |
| **プロへの報酬** | レッスン料の **80%** |

### 2.2 決済方式

| 項目 | 仕様 |
|-----|------|
| **決済手段** | クレジットカード（Stripe） |
| **対応カード** | Visa / Mastercard / AMEX / JCB |
| **課金方式** | ポイント事前購入制 |

## 5. 動画診断フロー

### 5.1 基本フロー

```
┌─────────────────────────────────────────────────────────┐
│  ユーザー                        プロ                   │
├─────────────────────────────────────────────────────────┤
│  1. プロ選択                                            │
│      ↓                                                  │
│  2. 動画撮影/選択                                       │
│     （MP4/MOV、500MB/2分以内）                          │
│      ↓                                                  │
│  3. 相談テキスト入力                                     │
│      ↓                                                  │
│  4. ポイント消費 ─────────→ 5. 依頼通知（プッシュ）     │
│                                    ↓                    │
│                              6. 動画確認                │
│                                    ↓                    │
│                              7. 回答動画撮影            │
│                                    ↓                    │
│  8. 回答通知 ←────────────  9. 回答送信                 │
│      ↓                        【48時間以内】            │
│  10. 回答閲覧                                           │
│      ↓                                                  │
│  11. 評価・レビュー投稿                                  │
└─────────────────────────────────────────────────────────┘
```

### 5.2 追加質問

| 項目 | 仕様 |
|-----|------|
| **回数** | 最大3回まで |
| **形式** | テキスト + 動画添付可 |
| **料金** | 追加質問ごとに別途ポイント消費 |

### 5.3 回答期限

| 項目 | 仕様 |
|-----|------|
| **期限** | 依頼から48時間以内 |
| **超過時** | 自動返金（ポイント返還） |

## 7. プロ審査フロー

### 7.1 審査プロセス

```
1. プロ登録申請
       ↓
2. 必要書類アップロード
   ├─ 身分証明書（運転免許証、パスポート等）
   └─ ゴルフ資格証明書（PGA資格、レッスンプロ資格等）
       ↓
3. 運営による審査（手動）
   ├─ 書類の真正性確認
   └─ 資格の有効性確認
       ↓
4. 審査結果
   ├─ 承認 → プロとして公開、サービス利用開始
   └─ 却下 → 理由通知、再申請可能
```

### 7.2 必要書類

| 書類 | 必須 | 備考 |
|-----|:----:|------|
| 身分証明書 | ○ | 運転免許証、パスポート、マイナンバーカード等 |
| ゴルフ資格証明書 | ○ | PGA資格、JGTO資格、各種レッスンプロ資格等 |

## 8. 技術要件

### 8.1 動画仕様

| 項目 | 仕様 |
|-----|------|
| **対応形式** | MP4 / MOV |
| **最大ファイルサイズ** | 500MB |
| **最大再生時間** | 2分 |
| **推奨撮影方向** | 横向き |
| **サーバー処理** | 必要に応じてMP4(H.264)にトランスコード |

### 8.2 プッシュ通知

| トリガー | 対象 | 通知内容 |
|---------|------|---------|
| 依頼受信 | プロ | 「新しい診断依頼が届きました」 |
| 回答受信 | ユーザー | 「プロから回答が届きました」 |
| 追加質問受信 | プロ | 「追加質問が届きました」 |
| 回答期限24時間前 | プロ | 「回答期限が近づいています」 |
| ポイント期限30日前 | ユーザー | 「ポイントの有効期限が近づいています」 |
| 審査完了 | プロ | 「審査が完了しました」 |

### 8.3 セキュリティ要件

| 項目 | 仕様 |
|-----|------|
| **決済情報** | アプリ側で保持せず、Stripeトークン決済を使用 |
| **個人情報** | SSL/TLS暗号化通信必須 |
| **認証** | JWT等のトークンベース認証 |
| **審査書類** | 暗号化保存、アクセス権限制御 |

## 9. キャンセル・返金ポリシー

### 9.1 自動返金

| ケース | 対応 |
|-------|------|
| プロが48時間以内に回答しない | 全額ポイント返還 |

### 9.2 手動返金（運営判断）

| ケース | 対応 |
|-------|------|
| 回答内容に重大な問題がある | 運営審査の上、返金判断 |
| システム障害による問題 | 全額返金 |

### 9.3 返金不可

| ケース | 理由 |
|-------|------|
| ユーザー都合のキャンセル | 依頼送信後のキャンセル不可 |
| 未使用ポイント | 規約により返金不可 |

## 11. 用語定義

| 用語 | 定義 |
|-----|------|
| **ユーザー** | ゴルフレッスンを受ける側（ゴルファー） |
| **プロ** | ゴルフレッスンを提供する側（レッスンプロ） |
| **動画診断** | ユーザーがスイング動画を送り、プロがアドバイス動画で回答する非同期サービス |
| **ポイント** | サービス内通貨（1pt = 1円） |
| **依頼** | ユーザーからプロへの動画診断リクエスト |
| **回答** | プロからユーザーへの診断結果（動画 + テキスト） |
